# Makrell# Packages

These packages are part of the Makrell language family for programming
languages, data formats, markup formats, and embedded DSLs.

Current `.NET` packages include:

- `MakrellSharp.Ast`
- `MakrellSharp.BaseFormat`
- `MakrellSharp.Mron`
- `MakrellSharp.Mrml`
- `MakrellSharp.Mrtd`
- `MakrellSharp.Compiler`
- `MakrellSharp.Cli`

The CLI package is distributed as a `.NET` tool and provides commands such as:

- `makrellsharp examples/hello.mrsh`
- `makrellsharp check examples/hello.mrsh`
- `makrellsharp build examples/hello.mrsh`
- `makrellsharp emit-csharp examples/hello.mrsh`

Project docs and family overview:

- Website: https://makrell.dev
- Repository: https://github.com/hcholm/makrell-omni
