Cookbook
========

The cookbook is where Makrell should feel useful quickly.

Unlike a tutorial, a cookbook is not primarily sequential.
It is a place to solve small, concrete problems:

* how to express a pipeline
* how to partially apply a function
* how to work with MRON or MRML
* how to use macros
* how to interoperate with a host ecosystem
