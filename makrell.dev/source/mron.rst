MRON
====

This page is kept as a compatibility entry point for readers arriving through
the older top-level ``mron`` path.

The main MRON documentation now lives under :doc:`mron/index`, where the format
has its own section with quick-start, syntax, and recipe pages.

What MRON is
------------

MRON is the Makrell family format for structured data. It is intended for cases
where you want a Makrell-shaped notation for configuration, data exchange, or
JSON-like content without adopting a full host language first.

If you are new to MRON, a practical route is:

#. start with the section overview
#. work through the quick start
#. use the syntax page as reference
#. use the cookbook pages for common data-shaping tasks
