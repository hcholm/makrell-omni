Installation
============

MakrellTS currently lives under ``impl/ts`` in the monorepo.

From ``impl/ts/``:

.. code-block:: bash

    bun install

Common commands
---------------

.. code-block:: bash

    bun run build
    bun run build:browser
    bun run test
    bun run typecheck
    bun run lint
    bun run test:browser

Environment focus
-----------------

MakrellTS is aimed at:

* Node.js
* Bun
* browser-oriented workflows
* JavaScript and TypeScript interop
