MRTD
====

This page is kept as a compatibility entry point for readers arriving through
the older top-level ``mrtd`` path.

The main MRTD documentation now lives under :doc:`mrtd/index`.

What MRTD is
------------

MRTD is the Makrell family tabular data format. It is intended for row-and-column
data that should stay simple and close to CSV in spirit while using Makrell-family
token rules.
