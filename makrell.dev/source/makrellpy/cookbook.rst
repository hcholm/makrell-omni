Cookbook
========

MakrellPy is one of the best places to learn the Makrell family by doing.

Useful recipe areas:

* partial application
* pipes and reverse pipes
* custom operators
* Python interop
* macros and quote/unquote
* embedded mini-languages
* pattern matching

Start with:

* :doc:`cookbook-functional`
* :doc:`cookbook-interop`
* :doc:`cookbook-macros`

And then deepen with:

* :doc:`functional`
* :doc:`interop`
* :doc:`metaprogramming`
