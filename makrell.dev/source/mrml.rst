MRML
====

This page is kept as a compatibility entry point for readers arriving through
the older top-level ``mrml`` path.

The main MRML documentation now lives under :doc:`mrml/index`, where the format
has its own section with quick-start, syntax, and recipe pages.

What MRML is
------------

MRML is the Makrell family format for structured markup. It is intended for
document fragments, pages, and other XML-like content expressed in Makrell
forms.

If you are new to MRML, a practical route is:

#. start with the section overview
#. work through the quick start
#. use the syntax page as reference
#. use the cookbook pages for fragments and page-shaped content
