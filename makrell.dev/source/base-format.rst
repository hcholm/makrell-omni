Makrell Base Format
===================

This page is kept as a compatibility entry point for readers arriving through
the older top-level ``base-format`` path.

The current MBF documentation now lives under :doc:`mbf/index`, where the
material is organised as a proper section rather than a single top-level page.

What MBF is
-----------

Makrell Base Format, usually shortened to MBF, is the structural foundation
shared across the Makrell family. It defines the core node shapes, bracket
forms, operator handling, and parsing model that the language and format layers
build on.

If you are new to MBF, the best way to approach it is:

#. read the section overview
#. work through the quick start
#. then use the syntax and parsing pages as reference material
